TecnologiaVersionUso
Java21 LTSLenguaje principal
Spring Boot3.2.4Framework web
Spring Data JPA3.2.xORM / Persistencia

Datos In-Memory vs Persistidos​

DatoAlmacenamientoRazon
Jugadores, Cartas, MazosMySQL (JPA)Datos permanentes
Partidas activasConcurrentHashMapBaja latencia en tiempo real
Salas del lobbyConcurrentHashMapEfimeras, no necesitan persistencia
Posiciones de jugadores (lobby 3D)WebSocket (memoria)Datos de sesion
Backup de mazosArchivo JSONProteccion ante reinicios
Catalogo de cartascards.json + MySQLSeed inicial desde JSON
