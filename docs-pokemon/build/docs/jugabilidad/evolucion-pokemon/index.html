📈 Evolución de Pokémon - Crece tu Poder

Guía completa del sistema de evolución

🔄 Cómo Funciona la Evolución​

Líneas Evolutivas​

Los Pokémon evolucionan en una cadena específica:

Etapa 0 (Básico)
    ↓
Etapa 1
    ↓
Etapa 2

Ejemplos reales:

Bulbasaur (Básico) → Ivysaur (Etapa 1) → Venusaur (Etapa 2)
Charmander (Básico) → Charmeleon (Etapa 1) → Charizard (Etapa 2)
Squirtle (Básico) → Wartortle (Etapa 1) → Blastoise (Etapa 2)

Requisitos para Evolucionar​

  1. Tener el Pokémon anterior en el Bench

    Para jugar Charmeleon, necesitas:
    ✅ Un Charmander en el Bench
    ✅ La carta de Charmeleon en mano

  2. Puede ser en el Bench

    Charmander está jugando (Bench)
    └─ Puedo evolucionarlo sin cambiar al campo activo

  3. Solo 1 evolución por Pokémon por turno

    Turno 1: Charmander (Básico) → Charmeleon (Etapa 1) ✅
    Turno 2: Charmeleon (Etapa 1) → Charizard (Etapa 2) ✅

    NO PUEDES:
    Turno 1: Charmander → Charmeleon → Charizard (3 en 1 turno) ❌

📋 Reglas de Evolución​

Timing​

Cuándo puedes evolucionar:

Turno 1: NO (Charmander jugado este turno - "restricción de nuevo jugado")
Turno 2+: SÍ (Turno anterior o anterior)

Restricción: "Jugado este Turno"​

Ejemplo:
┌─────────────────────┐
│  Turno 1            │
│  Juego Charmander   │
│  (Básico)           │
│  ❌ No puedo        │
│  evolucionarlo ahora│
└─────────────────────┘
        ↓
┌─────────────────────┐
│  Turno 2            │
│  Charmander ya está │
│  desde hace turno   │
│  ✅ Puedo           │
│  evolucionar a      │
│  Charmeleon         │
└─────────────────────┘

🎯 Estrategia de Evolución​

Ventajas​

✅ Más HP: Evolucionados suelen tener más salud ✅ Ataques mejores: Evolucionados tienen ataques más poderosos ✅ Habilidades especiales: Pueden tener efectos únicos ✅ Flexibilidad: Puedes evolucionar cuando sea conveniente

Desventajas​

❌ Recursos: Necesitas espacio en banco (máximo 5) ❌ Tiempo: Esperar turnos para evolucionar ❌ Pérdida de energía: Si derrotan Pokémon, pierdes energía y cartas

📊 Ejemplos en Batalla​

Escenario 1: Evolución Exitosa​

Turno 1:
└─ Juego Charmander básico (40 HP)

Turno 2:
├─ Charmander: Energía Fuego 1/1
├─ Evoluciono a Charmeleon
│  (usando carta de Charmeleon en mano)
└─ Charmeleon: 50 HP, ataque "Flame Burst" (40 dmg)

Turno 3:
├─ Charmeleon: Energía Fuego 2/2
├─ Evoluciono a Charizard
│  (usando carta de Charizard en mano)
└─ Charizard: 120 HP, ataque "Crimson Dive" (100 dmg)

Turno 4: ✅ ¡Ataco con Charizard full power!

Escenario 2: Evolución Interrumpida​

Turno 1: Juego Charmander (40 HP, Energía 0/1)
Turno 2:
├─ Oponente ataca: Squirtle hace 40 daño
├─ Charmander: KO (40 → 0 HP)
├─ Charmander al descarte
├─ Tomo 1 premio (rival)
└─ ❌ Nunca evolucioné

Lección: No siempre puedes esperar. A veces necesitas Pokémon básico fuertes.

💡 Tips Estratégicos​

:::tip Consejo 1: Línea Evolutiva Completa Siempre incluye 3-4 copias del Básico, 2-3 de Etapa 1, 1-2 de Etapa 2 en tu mazo.

Razón: Si pierdes uno, tienes copias extras. :::

:::tip Consejo 2: Pokémon Básico Secundarios No cuentas solo con tu línea evolutiva principal. Ten 2-3 Pokémon básicos para flexibilidad.

Razón: Si tu línea principal no aparece, aún tienes opciones. :::

:::tip Consejo 3: Timing de Evolución Evita evolucionar si tu Pokémon está a punto de ser atacado derrotado.

Razón: Perderías la carta de evolución también. :::
