# ⚡ Pokémon Trading Card Game - Documentación Oficial

> **"Gotta Document 'Em All!"** ✨

Bienvenido a la documentación completa y exhaustiva del **Pokémon Trading Card Game (TCG)**, un proyecto full-stack que implementa todas las mecánicas del juego de cartas más popular del mundo.

---

## 🎯 ¿Qué es este proyecto?

Este es un **Trading Card Game interactivo** desarrollado con:

- **Backend**: Spring Boot 3.2.4 + Java 21
- **Frontend**: React + TypeScript + Three.js
- **Base de Datos**: H2 (desarrollo) / PostgreSQL (producción)
- **Tiempo Real**: WebSocket para batallas multijugador

El proyecto implementa **todas las reglas del Pokémon TCG oficial** con:
- ♻️ Sistema de turnos exacto
- ⚡ Mecánicas de evolución
- 🔥 Efectos de tipos (Fuego, Agua, Planta, etc)
- 💥 Batallas turn-based en tiempo real
- 🎁 Sistema de sobres (booster packs)
- 🛠️ Constructor de mazos con validación
