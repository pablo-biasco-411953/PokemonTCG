# ✨ Efectos y Habilidades - Mecánicas Avanzadas

> Los efectos especiales que hacen el juego interesante

---

## 🎯 Tipos de Efectos

### Habilidades Pasivas
```
Siempre están activas.

Ejemplo: "Regenerador"
"Al final de tu turno, 
restaura 10 HP a este Pokémon."

Efecto: Automático cada turno
```

### Ataques
```
Requieren Energía y turno.

Ejemplo: "Flame Charge"
Costo: 1 Energía Fuego
Daño: 30
Efecto: +1 Energía a este Pokémon
```

### Efectos Secundarios
```
Daño al banco, descarte, etc.

Ejemplo: "Confusión"
50% de no poder atacar
```

### Efectos Persistentes
```
Estados que quedan en el Pokémon.

Tipos:
- Quemadura (10 HP por turno)
- Envenenamiento (10 HP por turno)
- Parálisis (50% de no atacar)
- Sueño (No ataca, moneda para despertar)
- Confusión (50% de no atacar)
```

## 📊 Estados Especiales

| Estado | Efecto | Cura |
|--------|--------|------|
| 🔥 Quemadura | -10 HP/turno | Poción |
| 💜 Envenenamiento | -10 HP/turno | Poción |
| ⚡ Parálisis | 50% no ataque | Full Heal |
| 😴 Sueño | Sin ataque | Despierta solo |
| 😵 Confusión | 50% no ataque | Full Heal |

## 💡 Estrategia

✅ Usa habilidades para ventaja continua  
✅ Planifica ataques con efectos secundarios  
✅ Cura estados cuando sea crítico  
❌ No bases todo en un estado (pueden curarlo)
