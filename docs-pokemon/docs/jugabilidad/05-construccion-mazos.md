# 🛠️ Construcción de Mazos - Crea tu Estrategia

> Guía para construir mazos válidos y competitivos

---

## ✅ Reglas Básicas

| Regla | Requisito |
|-------|-----------|
| **Total de cartas** | Exactamente 60 |
| **Máximo de copias** | 4 de la misma carta (excepto Energía) |
| **Mínimo energías** | 20 recomendado |
| **Máximo energías** | Ilimitado (pero limita Pokémon) |

## 📊 Distribución Típica (60 cartas)

```
Pokémon (12-15 cartas)
├─ 2-3 Básicos principales
├─ 1-2 Evoluciones
├─ 2-3 Pokémon secundarios
└─ Diluye el mazo si no necesitas línea completa

Energías (20-25 cartas)
├─ Energía primaria: 12-15
├─ Energía secundaria: 8-10
└─ Energía Incolora: 2-5

Entrenadoras (20-25 cartas)
├─ Objetos: 10-12
├─ Soportadores: 4-6
├─ Estadios: 1-2
└─ Otras estratégicas: 5-8
```

## 🎯 Arquetipos de Mazo

### Control (Anti-Agro)
```
Objetivo: Resistir y contraatacar
Características:
• Muchas cartas de curación
• Pokémon defensivos
• Bloqueos de oponente
```

### Aggro (Rápido)
```
Objetivo: Ganar rápido
Características:
• Pokémon con bajo costo de ataque
• Muchos ataques
• Poco control
```

### Combo (Estratégico)
```
Objetivo: Armar secuencia ganadora
Características:
• Pocas cartas repetidas
• Muchos efectos diferentes
• Requiere práctica
```

## 🚫 Errores Comunes

❌ Menos de 60 cartas (juego inconsistente)  
❌ 5+ copias de la misma carta (imposible)  
❌ Solo 10 energías (no puedes atacar)  
❌ Línea evolutiva incompleta (Charmander sin Charmeleon)
